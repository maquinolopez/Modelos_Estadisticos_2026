# Modelos Estadísticos — Maestría en Cómputo Estadístico (CIMAT)

Repositorio del curso **Modelos Estadísticos** de la **Maestría en Cómputo Estadístico** del **Centro de Investigación en Matemáticas (CIMAT)**.

Este repositorio contiene el material utilizado durante el curso, incluyendo diapositivas y tareas.

---

## 📚 Contenido del repositorio

El repositorio está organizado de la siguiente manera:

```
.
├── diapositivas/
│   ├── clase_01.pdf
│   ├── clase_02.pdf
│   ├── ...
│
├── tareas/
│   ├── tarea_01.pdf
│   ├── tarea_02.pdf
│   ├── soluciones/
│   ├── ...
│
└── README.md
```

### 📂 `diapositivas/`

Contiene las diapositivas utilizadas en cada sesión del curso.

---

### 📂 `tareas/`

Contiene las tareas asignadas durante el curso.

Cada tarea puede incluir:

- Enunciado en PDF

- Archivos de apoyo (datos o código)
